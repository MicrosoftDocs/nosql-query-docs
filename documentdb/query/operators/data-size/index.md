# Data size - operators

Data size operators are used to determine the size of data structures and values.

| | Description |
| --- | --- |
| [**`$binarySize`**]($binarysize.md) | The $binarySize operator is used to return the size of a binary data field. |
| [**`$bsonSize`**]($bsonsize.md) | The $bsonSize operator returns the size of a document in bytes when encoded as BSON. |
