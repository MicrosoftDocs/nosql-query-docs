# Object expression - operators

Object expression operators perform operations on objects and object properties.

| | Description |
| --- | --- |
| [**`$mergeObjects`**]($mergeobjects.md) | The $mergeObjects operator merges multiple documents into a single document |
| [**`$objectToArray`**]($objectToArray.md) | The objectToArray command is used to transform a document (object) into an array of key-value pairs. |
| [**`$setField`**]($setField.md) | The setField command is used to add, update, or remove fields in embedded documents. |
