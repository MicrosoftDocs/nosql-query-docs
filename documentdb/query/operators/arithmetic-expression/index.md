# Arithmetic expression - operators

Arithmetic expression operators perform mathematical operations on numeric values.

| | Description |
| --- | --- |
| [**`$abs`**]($abs.md) | The $abs operator returns the absolute value of a number. |
| [**`$add`**]($add.md) | The $add operator returns the sum of two numbers or the sum of a date and numbers. |
| [**`$ceil`**]($ceil.md) | The $ceil operator returns the smallest integer greater than or equal to the specified number. |
| [**`$divide`**]($divide.md) | The $divide operator divides two numbers and returns the quotient. |
| [**`$exp`**]($exp.md) | The $exp operator raises e to the specified exponent and returns the result |
| [**`$floor`**]($floor.md) | The $floor operator returns the largest integer less than or equal to the specified number |
| [**`$ln`**]($ln.md) | The $ln operator calculates the natural logarithm of the input |
| [**`$log`**]($log.md) | The $log operator calculates the logarithm of a number in the specified base |
| [**`$log10`**]($log10.md) | The $log10 operator calculates the log of a specified number in base 10 |
| [**`$multiply`**]($multiply.md) | The $multiply operator multiplies the input numerical values |
| [**`$pow`**]($pow.md) | The `$pow` operator calculates the value of a numerical value raised to the power of a specified exponent. |
| [**`$round`**]($round.md) | The $round operator rounds a number to a specified decimal place. |
| [**`$sqrt`**]($sqrt.md) | The $sqrt operator calculates and returns the square root of an input number |
| [**`$subtract`**]($subtract.md) | The $subtract operator subtracts two numbers and returns the result. |
| [**`$trunc`**]($trunc.md) | The $trunc operator truncates a number to a specified decimal place. |
