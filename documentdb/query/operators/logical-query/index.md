# Logical query - operators

Logical query operators combine boolean expressions using logical operations.

| | Description |
| --- | --- |
| [**`$and`**]($and.md) | The $and operator joins multiple query clauses and returns documents that match all specified conditions. |
| [**`$nor`**]($nor.md) | The $nor operator performs a logical NOR on an array of expressions and retrieves documents that fail all the conditions. |
| [**`$not`**]($not.md) | The $not operator performs a logical NOT operation on a specified expression, selecting documents that don't match the expression. |
| [**`$or`**]($or.md) | The $or operator joins query clauses with a logical OR and returns documents that match at least one of the specified conditions. |
