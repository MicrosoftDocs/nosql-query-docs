# Field update - operators

Field update operators are used to modify specific fields in documents during update operations.

| | Description |
| --- | --- |
| [**`$currentDate`**]($currentdate.md) | The $currentDate operator sets the value of a field to the current date, either as a Date or a timestamp. |
| [**`$inc`**]($inc.md) | The $inc operator increments the value of a field by a specified amount. |
| [**`$mul`**]($mul.md) | The $mul operator multiplies the value of a field by a specified number. |
| [**`$rename`**]($rename.md) | The $rename operator allows renaming fields in documents during update operations. |
| [**`$setOnInsert`**]($setoninsert.md) | The $setOnInsert operator sets field values only when an upsert operation results in an insert of a new document. |
