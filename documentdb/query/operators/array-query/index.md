# Array query - operators

Array query operators are used to query and filter array elements.

| | Description |
| --- | --- |
| [**`$all`**]($all.md) | The $all operator helps finding array documents matching all the elements. |
| [**`$elemMatch`**]($elemmatch.md) | The $elemMatch operator returns complete array, qualifying criteria with at least one matching array element. |
| [**`$size`**]($size.md) | The $size operator is used to query documents where an array field has a specified number of elements. |
