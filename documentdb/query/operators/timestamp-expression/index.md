# Timestamp expression - operators

Timestamp expression operators perform operations on timestamp values.

| | Description |
| --- | --- |
| [**`$tsIncrement`**]($tsincrement.md) | The $tsIncrement operator extracts the increment portion from a timestamp value. |
| [**`$tsSecond`**]($tssecond.md) | The $tsSecond operator extracts the seconds portion from a timestamp value. |
