# Literal expression - operators

Literal expression operators are used to represent literal values in aggregation expressions.

| | Description |
| --- | --- |
| [**`$literal`**]($literal.md) | The $literal operator returns the specified value without parsing it as an expression, allowing literal values to be used in aggregation pipelines. |
