# Date expression - operators

Date expression operators perform operations on date and time values.

| | Description |
| --- | --- |
| [**`$dateAdd`**]($dateadd.md) | The $dateAdd operator adds a specified number of time units (day, hour, month etc) to a date. |
| [**`$dateDiff`**]($datediff.md) | The $dateDiff operator calculates the difference between two dates in various units such as years, months, days, etc. |
| [**`$dateFromParts`**]($datefromparts.md) | The $dateFromParts operator constructs a date from individual components. |
| [**`$dateFromString`**]($datefromstring.md) | The $dateDiff operator converts a date/time string to a date object. |
| [**`$dateSubtract`**]($datesubtract.md) | The $dateSubtract operator subtracts a specified amount of time from a date. |
| [**`$dateToParts`**]($datetoparts.md) | The $dateToParts operator decomposes a date into its individual parts such as year, month, day, and more. |
| [**`$dateToString`**]($datetostring.md) | The $dateToString operator converts a date object into a formatted string. |
| [**`$dateTrunc`**]($datetrunc.md) | The $dateTrunc operator truncates a date to a specified unit. |
| [**`$dayOfMonth`**]($dayofmonth.md) | The $dayOfMonth operator extracts the day of the month from a date. |
| [**`$dayOfWeek`**]($dayofweek.md) | The $dayOfWeek operator extracts the day of the week from a date. |
| [**`$dayOfYear`**]($dayofyear.md) | The $dayOfYear operator extracts the day of the year from a date. |
| [**`$hour`**]($hour.md) | The $hour operator returns the hour portion of a date as a number between 0 and 23. |
| [**`$isoDayOfWeek`**]($isodayofweek.md) | The $isoDayOfWeek operator returns the weekday number in ISO 8601 format, ranging from 1 (Monday) to 7 (Sunday). |
| [**`$isoWeek`**]($isoweek.md) | The $isoWeek operator returns the week number of the year in ISO 8601 format, ranging from 1 to 53. |
| [**`$isoWeekYear`**]($isoweekyear.md) | The $isoWeekYear operator returns the year number in ISO 8601 format, which can differ from the calendar year for dates at the beginning or end of the year. |
| [**`$millisecond`**]($millisecond.md) | The $millisecond operator extracts the milliseconds portion from a date value. |
| [**`$minute`**]($minute.md) | The $minute operator extracts the minute portion from a date value. |
| [**`$month`**]($month.md) | The $month operator extracts the month portion from a date value. |
| [**`$second`**]($second.md) | The $second operator extracts the seconds portion from a date value. |
| [**`$week`**]($week.md) | The $week operator returns the week number for a date as a value between 0 and 53. |
| [**`$year`**]($year.md) | The $year operator returns the year for a date as a four-digit number. |
