# Element query - operators

Element query operators are used to query document elements based on their existence and type.

| | Description |
| --- | --- |
| [**`$exists`**]($exists.md) | The $exists operator retrieves documents that contain the specified field in their document structure. |
| [**`$type`**]($type.md) | The $type operator retrieves documents if the chosen field is of the specified type. |
