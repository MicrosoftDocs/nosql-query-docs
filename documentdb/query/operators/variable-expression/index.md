# Variable expression - operators

Variable expression operators are used to define and reference variables in expressions.

| | Description |
| --- | --- |
| [**`$let`**]($let.md) | The $let operator allows defining variables for use in a specified expression, enabling complex calculations and reducing code repetition. |
