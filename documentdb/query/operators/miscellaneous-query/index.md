# Miscellaneous query - operators

Miscellaneous query operators include various utility and helper operators for querying that don't fit into other categories.

| | Description |
| --- | --- |
| [**`$comment`**]($comment.md) | The $comment operator adds a comment to a query to help identify the query in logs and profiler output. |
| [**`$natural`**]($natural.md) | The $natural operator forces the query to use the natural order of documents in a collection, providing control over document ordering and retrieval. |
| [**`$rand`**]($rand.md) | The $rand operator generates a random float value between 0 and 1. |
